## gulp

* 压缩图片

* 自动刷新浏览器

* 编译less／sass

  ### 依赖安装

  * node.js环境


*   全局安装gulp

  * 在项目本地安装gulp

  * 本地安装相关插件

    * npm intall gulp-imagemin —save -dev

    * npm install browser-sync —save -dev

    * npm install gulp -less —save -dev

    * npm install gulp-sass —save -dev

    * ...
